局域网共享代理

JiangXiaoyu

·

2023/02/07

在很多时候，局域网内实现共享代理是很重要的，比如刷欧版系统后的启动验证，还有绕过代理服务商的用户数量限制，实现多台机器共用同一个账户。

第一步，PC 或 Android 设备的 V2rayN 或其他代理软件上打开允许局域网设置

第二步，需要用网的 Android 机的 wifi 高级设置，Proxy 选项

第三步，选择手动，填写第一步中设备的 IP 地址，和 V2rayN 上的 http 端口号

完工
